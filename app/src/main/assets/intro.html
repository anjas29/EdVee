ED-V (VOCATIONAL EDUCATION): MEDIA PEMBELAJARAN SENSOR DAN AKTUATOR BERBANTUAN INTERNET OF THINGS (IOT) DENGAN GRAPHICAL USER INTERFACE (GUI) MENGGUNAKAN SMARTPHONE ANDROID DALAM PENINGKATAN KOMPETENSI LULUSAN PENDIDIKAN VOKASIONAL BIDANG KEAHLIAN OTOMASI INDUSTRI
Sasaran:
ED-V merupakan aplikasi perangkat lunak yang bekerja pada smartphone Android. ED-V ditujukan untuk pelajar, pendidik, khususnya bidang pendidikan vokasi, program keahlian elektronika, informatika, hingga otomasi industri.

Manfaat:
ED-V memiliki beberapa keunggulan utama yaitu, 1) sebagai modul pembelajaran sensor dan actuator melalui kendali terprogram berbasis Internet of Things, 2) sebagai buku pedoman belajar pemprograman Bahasa C yang digunakan untuk membangun embedded system, 3) menggunakan aplikasi ED-V sebagai simulasi embedded system berbasis IoT.

Deskripsi:
Indonesia merupakan Negara terbesar keempat di dunia dengan jumlah penduduk mencapai 249,9 juta pada 2013 (Bank Dunia, 2013). Persaingan bursa tenaga kerja juga semakin meningkat, sedangkan pemberlakuan pasar bebas ASEAN telah dimulai. Sumber daya manusia (SDM) Indonesia harus mampu berkompetisi dalam dunia internasional. Namun, kesadaran masyarakat tentang pendidikan masih kurang. Menurut data dari kelurahan Kota Baru perjanuari 2017, hanya 2 orang yang melanjutkan sampai ke S1 dan 3 orang yang melanjutkan kejenjang Diploma. Sumber daya manusia harus diciptakan dan dilatih untuk mempunyai kompetensi yang baik agar bisa bersaing dalam dunia global. Internet of things (IoT) merupakan teknologi yang memungkinkan segala sesuatu dapat dikendalikan melalui koneksi internet. Sehingga diperlukan suatu upaya untuk menyiapkan SDM yang mampu menguasai teknologi IoT melalui media pembelajaran berbasis teknologi. Penelitian ini menggunakan metode Research and Development (RnD). ED-V sebagai media pembelajaran sensor dan actuator menggunakan kendali terprogram berbasis IoT telah diujikan dan dinyatakan layak tanpa syarat.

ED-V for our future vocational education:
ED-V merupakan media pembelajaran sensor dan actuator menggunakan kendali terprogram berbasis IoT. ED-V memiliki 3 sub system utama yaitu software android, hardware dan system transmisi data berbasis IoT. ED-V dapat menjadi bentuk miniature simulasi sistem otomasi industri berbasis IoT. ED-V dilengkapi perangkat belajar berupa sensor LDR, Sensor SR-04, Relay, WiFi Module ESP8266, dan chip mikrokontroller ATMega328 sebagai hardware pelengkap. ED-V merupakan dilengkapi dengan fitur interfacing melelaui smartphone Android, dan modul pembelajaran materi teknologi otomasi industri dan IoT yang terdapat dalam aplikasi ED-V.

Fitur:
Fitur: • Belajar: Menu ini berisi opsi materi belajar berupa materi Sensor, actuator, kendali terprogram, dan Internet of Things
• Modul Pembelajaran: Merakit IoT embedded system, yang berisi tentang tutorial merakit embedded system berbasis IoT menggunakan ED-V.
• Coba ED-V IoT: Aplikasi interfacing dan monitoring data sensor berbasis IoT.
